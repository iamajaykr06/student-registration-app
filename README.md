# Student Registration & Search App

HTML + Tailwind CSS + PHP + MySQL lab assignment.

## Setup
1. Import `schema.sql` in MySQL Workbench
2. Create `db_connect.php` with your credentials (excluded from Git)
3. Run `php -S localhost:8000` inside the project folder
4. Open http://localhost:8000

## Features
- Register students — POST method → register.php
- Search by Student ID — GET method → search.php
